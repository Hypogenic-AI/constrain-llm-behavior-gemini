# Constraining LLM Behavior via Abstention

This project investigates methods to motivate LLMs to abstain from answering when uncertain.

## Quick Start

1.  **Read the Summary**: See [SUMMARY.md](SUMMARY.md) for a project overview.
2.  **Explore Resources**: Check [resources.md](resources.md) for gathered papers and data.
3.  **Run Experiment**:
    ```bash
    make install
    make analyze  # Check dataset stats
    make run
    make plot
    make inspect  # View specific examples
    ```

For detailed experiment instructions, see [experiment/README.md](experiment/README.md).
For a file manifest, see [MANIFEST.md](MANIFEST.md).
For common issues, see [TROUBLESHOOTING.md](TROUBLESHOOTING.md).
